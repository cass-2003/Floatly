# DeskLite

轻量级 Windows 桌面小组件 — 常驻桌面的一小块「今日信息条」：时钟、农历、黄历、天气、待办、倒数日，不占地方、低资源占用。

## 技术栈

- **.NET 8** + **WPF**（无边框透明窗口）
- **Windows Forms** 系统托盘
- **Open-Meteo** 天气（免费、无需 API Key）
- **lunar-csharp** 农历 / 黄历计算
- 单项目、单 exe，本地 JSON 存储，**1 个 NuGet** 依赖

## 已实现功能

### 时钟与历法

- 公历日期、星期；12 / 24 小时制，秒显示可开关
- 农历月日、节气、生肖、干支（简洁一行，或切换为黄历详情）
- **黄历详情**（51 万年历风格布局）：宜忌、五行、冲煞、值神、建除、吉神凶神、彭祖百忌、二十八宿、时辰吉凶、当前时辰卡片；支持前后翻日
- **周历 / 月历**切换，翻页、回到今天、跳转日期
- **2026 年节假日**：周历 / 月历格子上标注 **休** / **班**（离线内置）

### 天气

- 当前温度、天气状况、今日高低温
- 日出日落、明日天气（可单独开关）
- 手动填城市或 **自动定位**（启动时 + 每 24 小时刷新）

### 今日清单

- 今日待办增删勾选，可选时间前缀
- **置顶** 1 条（星标排序优先）
- 主面板最多显示 5 条
- 待办到时 **托盘 Toast 提醒**
- 全局快捷键：`Ctrl+Shift+N` 快速添加待办
- *进行中*：`TodoListWindow`（全部 / 进行中 / 已完成、搜索、编辑）已在代码中实现，尚未接入主界面入口（未提交）

### 可选模块（设置中开关与排序）

- **倒数日**（内置节日 + 自定义事件，带进度条）
- **年进度**（全年已过百分比与进度条）
- **每日一句**（离线语录，按日随机）
- **速记便签**（一行 scratch，写入本地数据）
- **模块显示顺序**：设置 → 模块 → 上移 / 下移调整桌面排列

### 窗口与外观

- 无边框透明面板，可拖拽移动
- **四边与四角拖拽调整大小**
- 深色 / 浅色主题；设置内 **透明度滑块**（30–100%）、**字号滑块**（10–16 pt）
- 窗口置顶、鼠标穿透、开机自启
- 右键菜单：快捷切换主题、透明度预设、日历视图
- 内容区可滚动，**滚动条隐藏**
- 全局快捷键：`Ctrl+Shift+D` 显示/隐藏

### 设置与托盘

- **设置窗口**（分栏：常规、外观、天气、日历、模块）集中管理配置
- 精简托盘菜单：显示/隐藏、设置、添加待办/倒数日、日历跳转、**导出数据备份**、退出
- 双击托盘图标显示/隐藏主窗口

## 运行与构建

需要 [.NET 8 桌面运行时](https://dotnet.microsoft.com/download/dotnet/8.0)。

```powershell
cd DeskLite
dotnet run
```

发布 Release（框架依赖，体积约 2–5 MB）：

```powershell
cd DeskLite
dotnet publish -c Release
```

输出目录：`DeskLite\bin\Release\net8.0-windows\publish\`

## 数据位置

| 文件 | 说明 |
|------|------|
| `%AppData%\DeskLite\settings.json` | 窗口位置、主题、模块开关与顺序、城市等偏好 |
| `%AppData%\DeskLite\data.json` | 待办、倒数日、速记便签等业务数据 |

托盘菜单 **导出数据备份** 可将 `data.json` 另存到桌面（`desk-lite-backup-日期.json`）。

## 托盘菜单

| 菜单项 | 说明 |
|--------|------|
| 显示/隐藏 | 切换主窗口可见性 |
| 设置... | 打开设置窗口 |
| 添加待办 | 弹出输入框快速添加 |
| 添加倒数日... | 新建自定义倒数事件 |
| 日历 → 回到今天 / 跳转日期... | 日历导航 |
| 导出数据备份 | 备份 `data.json` 到桌面 |
| 退出 | 关闭应用 |
