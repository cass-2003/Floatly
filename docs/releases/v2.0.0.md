# Floatly v2.0.0 Release Notes

## 🎨 重大更新：Glassmorphism UI 2.0

Floatly 2.0 带来全新的毛玻璃设计语言，灵感来自 Windows 11 Fluent Design 与 Arc Browser，提供更现代、更轻盈的桌面体验。

---

## ✨ 新增功能

### 🎭 **皮肤系统**
- **视频背景皮肤**（实验性）：支持选择 `.mp4` / `.mov` / `.avi` 等视频格式作为背景
  - ⚠️ **当前状态：功能暂不可用**
  - 由于播放引擎存在严重 bug，**v2.0.0 中视频背景无法正常播放**
  - 症状：选择视频后黑屏、无画面、或直接崩溃
  - **请勿在 v2.0.0 中使用视频背景功能**
  - 该功能将在 v2.1.0 完全重写播放引擎后修复
- 图片背景自定义：支持 `.jpg` / `.png` / `.bmp` 等格式
- 图片遮罩透明度调节（0–85%），提升文字可读性
- 纯色背景模式

### ⌨️ **可自定义快捷键**
- 自定义「显示/隐藏」快捷键（默认 `Ctrl+Shift+D`）
- 自定义「快速添加待办」快捷键（默认 `Ctrl+Shift+N`）
- 支持录制模式实时捕获按键组合

### 🎨 **字体与颜色**
- **字体颜色自定义**：6 色预设 + 自定义 #RRGGBB 十六进制颜色
- **字体选择器**：下拉列表 + 手动输入已安装字体名称
- **字号滑块**：10–16 pt 精细调节

### 💼 **工作效率模块**
- **下班倒计时**：自定义上下班时间，工作日模式，实时日进度条
- **摸鱼小助手**：按月薪、工作天数、工作时长实时计算收入，金色字体每秒跳动显示
- **工作时间配置**：上班时间、下班时间、每月工作天数、每日工作小时

### 📝 **速记便签增强**
- 多条便签管理（最多 20 条）
- 独立编辑窗口
- 置顶功能
- 颜色标签分类
- 搜索功能
- 一键复制

## 📋 已知问题

### ⚠️ 视频背景功能无法使用（P0 严重 Bug）

**问题描述**：  
设置中的「视频背景」选项虽然可见，但**功能完全不可用**。选择视频文件后会出现：
- 窗口黑屏无画面
- 程序无响应或崩溃
- 视频文件无法被识别

**根本原因**：  
MediaElement 播放引擎存在严重兼容性问题，当前实现方案不可行。

**影响范围**：  
仅影响视频背景功能。图片背景、纯色背景、默认毛玻璃模式均正常工作。

**规避方法**：  
- ✅ **请使用图片背景**（完全可用，推荐 .jpg / .png 格式）
- ✅ 使用默认毛玻璃效果
- ✅ 使用纯色背景模式
- ❌ **切勿选择视频背景**

**修复计划**：  
- v2.1.0 将完全重写视频播放引擎
- 考虑迁移至 Windows.Media.Playback 或 FFmpeg 方案
- 预计 2026 年 6 月下旬修复

## 📦 系统要求

- **操作系统**：Windows 10 1809+ / Windows 11
- **运行时**：.NET 8 Desktop Runtime (x64)
  - 下载：https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0
- **硬件**：
  - 最低配置：2GB RAM / 集成显卡
  - 推荐配置：4GB RAM（用于图片背景 + 多模块）
