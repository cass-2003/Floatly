## Floatly v2.0.22

### 新增
- **主小组件 2.0 布局**：围绕设计稿重构桌面小组件，加入天气、黄历、待办、番茄钟、下班倒计时、薪水助手、每日一句和日历信息流。
- **设置页 2.0**：重做设置页卡片排布，支持更紧凑的窗口行为、工作时间、时钟显示、快捷键、摸鱼收入、天气、日历和主题配置。
- **多级天气定位兜底**：自动定位失败时会结合定位服务、IP 识别、经纬度反查和手动城市，尽量避免只显示经纬度。
- **桌面缩放交互**：支持四边四角拖动缩放，默认尺寸按设计稿比例缩小，避免小组件变成巨无霸。

### 改进
- **毛玻璃视觉升级**：增强主窗口 Acrylic/Blur 背板、半透明卡片、玻璃边框和层次阴影，更接近设计图的毛玻璃风格。
- **图标系统替换**：接入新的图标资源包，替换天气、日出日落、设置、固定、薪水和模块图标。
- **主题配置收紧**：优化字体、字号、字体颜色、窗口透明度、背景遮罩透明度、皮肤系统和窗口大小区域的排版密度。
- **日历与天气卡片重排**：天气卡片更宽、日历卡片更紧凑，定位状态行按设计稿样式展示。
- **小组件密度优化**：缩小顶部时间字号和模块间距，调整番茄钟、黄历、待办、薪水助手、每日一句等卡片比例。
- **版本同步**：安装包、程序集、应用内版本号和构建脚本统一到 `2.0.22`。

### 修复
- **修复背景图直角溢出**：下班倒计时、今日已赚薪水、每日一句卡片的背景图现在会按卡片圆角绘制。
- **修复输入框遮挡**：优化设置页颜色、薪水、城市、字体等输入区域，减少图标或单位遮住文字的问题。
- **修复手动城市不生效**：手动输入城市会同步更新解析城市并触发天气刷新。
- **修复进度条颜色丢失**：恢复倒计时、年度进度、下班倒计时等进度填充颜色。
- **修复设置页中文/布局问题**：清理部分编码异常和卡片高度、输入框高度、行距不一致问题。

### 安装说明
- 需要 [.NET 8 桌面运行时（x64）](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)。
- 推荐下载 **Floatly-Setup-2.0.22.exe**（简体中文安装向导）。
- 绿色版：**Floatly-2.0.22-win-x64.zip**（解压后运行 `Floatly.exe`）。

### 升级说明
- 覆盖安装即可，用户配置保存在 `%AppData%\Floatly\`。
- 如果已打开旧版本 Floatly，建议先退出旧进程后再安装，避免 Windows 文件占用导致安装包替换失败。
